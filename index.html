Description

This site shows the different ways in which the 'anchor' element can be used. This includes external links, which constitute links to websites of entirely different domains, as well as internal links, which may direct the user to different sections of text on a page, or to a different page on the site.

At its core, the anchor or <a> element is used to create a hyperlink. The destination is supplied via a mandatory 'href' attribute, which tells the browser where to direct the user. This can take the form of a URL for external websites, an 'id' of another element for navigation within the same web page, or another .html file for navigation between the different pages of the site. It can even be used to open a file in full screen given an 'iframe' element.

Hyperlinks in HTML can be further customized with additional attributes. For instance, the target attribute can define how the link is opened, with target="_blank" making the browser display the destination of the link in a new tab.

Internal Links: Jumping to a Section

The anchor tag can jump between the different sections of a web page by placing an "id" attribute within the 'href' atrribute of the anchor tag. The syntax is: 

                    <a href="#idName"> [Link Text] </a>
with 'idName' being replaced with the actual name of the element's id.

External Links: Same vs. New Tab

With external links, an absolute path---typically the full URL---is needed in the anchor tag's 'href' attribute.

For that same example to open in a new tab, we need to add the 'target' attribute and set it to '_blank', as follows:

  This is a link to the Mozilla documentation on the anchor tag (opens in a new tab)

The syntax for this may look like: 

                <a href="[URL]" target="_blank" rel="noopener" rel="noreferrer"> 
                   [Link Text] 
                </a>

The 'rel' attributes are important to include for security reasons, though some modern web browsers will automatically add them whenever 'target' is set to '_blank'.
